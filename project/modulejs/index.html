modulejs

modulejs is a modular and Object-Oriented javascript framework that is designed to write a high maintainability and reusability of the javascript code.

Features

  • HTML document query/traversing/manipulation
  • Lightweight selector engine
  • Event handling
  • Animation
  • Cookie/localStorage storage
  • Language extensions
  • Ajax/JSONP request
  • Template engine
  • Rich utility class

modulejs is compatible with Internet Explorer 6+, Firefox 2+ (and browsers based on gecko) Chrome 4+, Safari 3+ and Opera 9+.

modulejs quick example:
var ajax = module("net.Ajax").ajax;
ajax({
	url:"localhost/ajaxdemo.php",      //be submited server-side address
	form:"formName",                   //be submited form's name
	timeout: 200,                      //timeout 
	success: function(response){       //success callback
		console.log(response);
	}
});

Note: This guide is only applicable to modulejs 0.1.0

module

"everything is module" that is an important concept in modulejs.

An example declare a module could look like:

module("lang.Array",function(){
	//IMPORT
	//...
	function trim(str){
		//...	
	}
	//EXPOSE
	return {
		"trim":trim
	};
});
